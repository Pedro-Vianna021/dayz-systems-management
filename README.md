dayz-systems-management
Projeto de estudo focado em Gerenciamento de Configuração (XML) e Otimização de Sistemas. Utilizo o ambiente de servidor do DayZ como laboratório para aplicar conceitos de Infraestrutura, lógica de recursos e automação.

DayZ Server: Central Economy & Systems Management
Este projeto consiste no desenvolvimento, configuração e manutenção de um servidor do jogo DayZ, focado no estudo de Configuration Management (XML), Infrastructure as Code (IaC) e Automação de Sistemas.

🛠️ Tecnologias e Conceitos Aplicados

• Lógica de Configuração: Manipulação de arquivos estruturados (XML/JSON) para balanceamento de sistemas.

• Gestão de Infraestrutura: Configuração de parâmetros de performance, latência e ciclos de vida de processos (Server Lifecycle).

• Versionamento: Uso de Git para controle de alterações e Rollback de configurações de economia.

• Integração de Sistemas: Utilização de Webhooks para monitoramento e logs em tempo real via Discord.

📈 Desafios Técnicos Superados

• Balanceamento de Economia (Central Economy): Definição de algoritmos de spawn baseados em `nominal`, `min` e `restock`.

• Otimização de Performance: Ajuste de `lifetime` de objetos para redução de carga no servidor e melhoria de FPS.

• Desenvolvimento Mobile: Gestão e edição de arquivos via ambiente mobile, demonstrando resiliência e adaptabilidade técnica.
